# 📊 Relatório de Faturamento e Vendas Weekly

Este projeto é uma ferramenta prática voltada para a automação comercial e análise de dados de varejo. O objetivo principal é transformar registros brutos de vendas em informações estratégicas para tomada de decisão.

## 🎯 Funcionalidades
- **Cálculo de Faturamento:** Processa o faturamento diário de uma operação ao longo dos 7 dias da semana.
- **Métricas Semanais:** Calcula automaticamente a média de faturamento do período.
- **Análise de Performance:** Identifica e exibe de forma clara quais dias superaram as expectativas e ficaram acima da média estipulada.

## 🛠️ Tecnologias e Conceitos Aplicados
- **Linguagem:** Java
- **Conceitos:** Estruturas de repetição (Loops), vetores/listas para armazenamento de dados e lógica matemática aplicada a negócios.

## 💻 Como Executar o Projeto
1. Clone este repositório ou baixe os arquivos.
2. Certifique-se de ter o compilador/interpretador da linguagem instalado.
3. Execute o arquivo principal através do terminal ou de sua IDE de preferência.
